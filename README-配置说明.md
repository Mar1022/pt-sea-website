# 星铂再生官网 - 内容配置说明

> 本文件说明如何修改网站内容，无需任何技术背景！

---

## 📁 修改内容前必读

### 文件位置
- 配置文件：`data.json`
- 图片文件夹：`/` 根目录（与 index.html 同级）

### 修改规则
1. 所有文字内容在 `data.json` 中修改
2. 图片直接替换同名文件
3. 修改后保存文件，刷新网站即可看到效果
4. **注意**：引号必须是英文引号 `""`，不能用中文引号 `""`

---

## 🔧 常用修改示例

### 1️⃣ 修改公司电话
找到：
```json
"电话": "0577-61586877",
```
改成：
```json
"电话": "0577-88888888",
```

### 2️⃣ 添加新的环保公示
在 `环保公示` 数组中添加新条目：
```json
{
    "日期": "2026-05-20",
    "类型": "monitoring",
    "类型名称": "环境监测",
    "标题": "2026年第一季度检测报告",
    "内容": "检测结果显示各项指标均符合标准..."
}
```

### 3️⃣ 修改环保公示类型说明
- `monitoring` = 环境监测
- `permit` = 许可证/应急预案
- `report` = 检测报告

### 4️⃣ 替换产品图片
1. 准备好新图片
2. 命名为 `ceramic-powder.jpg`
3. 替换网站根目录下的 `ceramic-powder.jpg`

## ⚠️ 注意事项

1. **不要修改 JSON 结构** - 只改文字内容和数字
2. **日期格式** - 使用 `年-月-日` 格式，如 `2026-05-15`
3. **备份** - 修改前建议先备份原文件
4. **格式检查** - 确保每个字符串用英文引号包裹

## ❓ 常见问题

**Q: 为什么修改后没变化？**
A: 请按 Ctrl+F5 强制刷新浏览器缓存

**Q: 图片显示不对？**
A: 检查图片文件名是否完全一致（包括大小写）

**Q: 网页打不开？**
A: 检查 data.json 格式是否正确，可用 https://json.cn/ 在线验证
